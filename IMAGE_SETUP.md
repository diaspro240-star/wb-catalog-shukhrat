# Image Setup Instructions

## Product Images Location

Your product images should be placed in the following directory:

```
public/images/products/
```

## Required Image Files

Replace the placeholder images with your actual product photos:

1. **Product 1 (Oversized Set - 46-60)**
   - File name: `wb-001.png`
   - Full path: `public/images/products/wb-001.png`
   - This corresponds to the first image you provided

2. **Product 2 (Sporty Elegance Set - 42-60)**
   - File name: `wb-002.png`
   - Full path: `public/images/products/wb-002.png`
   - This corresponds to the second image you provided

## How to Replace

1. Navigate to the `public/images/products/` folder in your project
2. Delete or replace the existing `wb-001.png` and `wb-002.png` files
3. Drag and drop your actual product images into this folder
4. Make sure the file names match exactly: `wb-001.png` and `wb-002.png`

## Image Format

- Format: PNG (recommended) or JPG
- Recommended dimensions: At least 1200px wide for high quality display
- Aspect ratio: Portrait orientation (3:4 ratio works best)

## Note

The images will be automatically optimized by Next.js for best performance across all devices.
